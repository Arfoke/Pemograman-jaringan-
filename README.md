# Deteksi IP Address (Java)

Program Java sederhana untuk **mendeteksi IP Address** dari input berupa **URL atau IP**, kemudian menampilkan:
- IP Address hasil resolusi DNS
- Kelas IP (A, B, C, D, E)
- Representasi IP dalam bentuk **biner (8-bit per oktet)**
